Recommendations

Other extensions

  • Vimium : Provides better keyboard navigation. You can even customize the shortcuts to better match the one true editor.
  • Lastpass : Automatically generate, remember, and fill in random passwords.
  • Greplin : Search your data.
  • QuickShift : Move Chrome tabs to a different window or to a new location within the current window using your keyboard.
  • Minimalist for Facebook
  • Minimalist for Google Calendar
  • Minimalist for Gmail
  • Rapportive : Replaces Gmail right-hand sidebar with information about who you are corresponding with.
